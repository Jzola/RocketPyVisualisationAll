Graph Creator Menu Script

UMLClass «Script»
RocketVisualisation UML :: Scripts :: «Script»Graph Creator Menu Script

Description

Used to to set up the UI of the Graph Creator Menu Canvas (filling drop-down lists with data), set up listeners for user controller actions, and accesses the Graph Creator component to actually generate the graphs.

Attributes

Visibility Name Type Description
public gCreator GraphCreator

Accesses the graph Creator object by a reference in the Unity Inspector. This allows the use of the GraphCreator script methods like CreateGraph as well as access to the variables inherited from Graph Common, such as variable lists and available outputs.

public xaxisDropdown Dropdown

Used to control the x axis variable on the created graph. Is set up during the start() method with variables obtained from gCreator.

Is a public variable so that the Inspector can be used to connect the Drop down game object to the script.

Operations

Visibility Name Description
public Start()

Sets up all of the drop-downs, and configures them will listeners. Attaches listeners to all buttons (except for a hide/minimizer button, which is handled separately by a prefab Show/maximize button with it's own script)
public changeGraphType()
public manualDimensionChange()
public dimensionChanged()

if the dimension of the graph is 2D, hide the dropdown for the z axis and disable it.
public dropDownTest()

Tests that the dropdowns can be changed via the inspector in desktop mode
private variableDropdownItemSelected()

The input variable has been changed. Store it to be used in graph creation.
private axisDropdownItemSelected(axisDropdown: Dropdown)

Listener used for all axis dropdowns. Will determine which dropdown was changed, will change the relevant axis to the matching variable.
public graphTypeChanged()

listener that disables the z axis options when the type of graph is 'bar'. Reenables if the type is 'scatter'
public createGraph()

Checks all the attributes to determine the graph creator values, then spawns a graph. Note the function will not be available if the button this operation is linked to has been disabled.
public Update()

Checks if the user should be able to create any more graphs.
public allowGraphCreation()

Checks if the user should be able to create a graph.
