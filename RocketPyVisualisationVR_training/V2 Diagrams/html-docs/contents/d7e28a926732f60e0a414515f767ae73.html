spawnCirclePrefab

UMLAttribute
RocketVisualisation UML :: Scripts :: «Script»GraphCreator :: spawnCirclePrefab

Description

A Game object that provides a visual cue as to where the next graph will spawn. Default is for graphs to spawn in a circle around the user.
