Description

The Main Scene holds all of the top level / parent components required to run the simulation. The VRMainCamera and EventSystem are the game objects that handle user movement and interaction with the scene. The MainMenuAnchor is the anchor point for the GraphMenu component (used to allow the user to create new graphs) The GraphCreator is a hidden object that holds a reference to the GraphCreator script, which is where all the configurations are used to generate specific types of graphs.
