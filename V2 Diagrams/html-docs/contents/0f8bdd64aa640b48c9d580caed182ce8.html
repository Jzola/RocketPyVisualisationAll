Description

The GraphMenuSet is based on a graph prefab and is only generated at runtime. It contains several canvas objects for various menus, and the Graph itself along with associated axis and trajectory information. It is created by the GraphCreator game object by calling the createGraph function from it's script component. DeveloperComponents: GraphCreator holds the script and component information for this game object. The older version of this component had both bar graph and scatter graph by default.
