GraphSpawnHandler

UMLClass «Script»
RocketVisualisation UML :: Scripts :: «Script»GraphSpawnHandler

Description

A new script created to help assign graphs to positions around the user. The spawn handler keeps track of existing graphs, potential spots, and used spots. It will also reposition graphs if a graph is deleted via graphConfig.
