GraphConfigCanvasMenu

UMLClass «GameObject»
RocketVisualisation UML :: ComponentsGameObjects :: «GameObject»GraphConfigCanvasMenu

Description

This menu contains all buttons in the user interface, and holds a script to control the interactable objects. The menu is generated when a scatter graph is created during runtime. It controls the settings of the graph by accessing the values in the Graph Config object
