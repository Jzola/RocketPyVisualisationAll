MaximizeCanvas{Prefab}

UMLClass «GameObject: Prefab»
RocketVisualisation UML :: ComponentsGameObjects :: «GameObject: Prefab»MaximizeCanvas{Prefab}

Description

There are several Maximise canvas objects connected to the graphs. Their purpose is to allow hiding of menus and displays. To work properly they must be either a sibling (on the same level) of the canvas that is to be minimised, or not connected to it at all, otherwise minimising the parent (the canvas to hide) will minimise the child ( the minimizer button) as well
